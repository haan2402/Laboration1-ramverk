Bootstrap CSS-Ramverk

Om Bootstrap

Länk till Bootstraps webbplats

Bootstrap är ett komponentbaserat ramverk som har många färdiga UI-komponenter. Ramverket innehåller färdiga komponenter som exempelvis knappar, formulär, tabeller och navbars. Bootstrap är ett poplulärt ramverk och är bra för att skapa responsiva och strukutrerade webbplatser. Fördelar med detta ramverk är att det finns mycket dokumentation om raverket samt snabb prototyping. Nackdelar är att det finns så mycket färdigt så det kan vara svårt att anpassa designen.

Utvärdering av Bootstrap

Jag har tyckt att det gått bra att arbeta med Bootstrap. Jag installerade via CDN på deras webbplats där jag infogade en link och en script länk. Dessa gjorde att jag kom åt alla komponenter genom att lägga till olika klasser i HTML koden. Det fanns många olika alternativ för att skapa den design man önskar, och fanns olika alternativ för att göra sidan responsiv.

Jag hade lite svårt i början att förstå hur alla klasser funkade tillsammans, men eftersom den fanns bra med dokumentation så gick det att få mycket tips därifrån. Jag tyckte det är ett smidigt ramverk, och med ganska lite kod kunde man göra stora skillnader på sidan.
