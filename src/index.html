CSS-Ramverk

Laboration 1

Syftet med laboration 1 är att bygga en webbplats där det finns två olika undersidor med två olika CCSS-ramverk. Jag kommer använda mig av Bootstrap och Tailwind för att lösa denna uppgift. Undersidorna ska dra nytta av de funktioner som ramverken har och kommer att innehålla tabeller, formulär och knappar, detta ska stylas med hjälp av ramverken.

Publicering

Jag valde att publicera min webbplats genom Github-repo och Netlify.

Jag skapade ett repository på min Github där jag pushade alla mina filer till repot. Jag skapade en .gitignore-fil där jag la till node_modules, detta för att inte behöva publicera alla de filerna. Sedan kopplade jag mitt Github-repo till Netlify där jag hade ett konto sedan tidigare, och eftersom jag endast arbetat med HTML-filer och CSS-filer behövde jag inte köra någon build. När jag skulle publicera på Netlify så angav jag src som publiceringsmapp, då genererade Netlify en länk till min webbplats där alla länkarna och innehåll fungerar som det ska.

Fördelar med Netlify är att man enkelt kan ladda upp sina filer till Github och sedan koppla det rpot till Netlify och få webbplatsen live, samt att man kan publicera gratis till en viss begränsning. Nackdelar med Netlify är att gratisversionen har en viss begränsning, för större projekt behöver man betala för att få bättre byggtid. Men sammanfattningsvis så är det ett smidigt sätt att publicera en webbplats för mindre projekt.
