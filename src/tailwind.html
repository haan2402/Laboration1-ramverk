Tailwind CSS-Ramverk

Om Tailwind

Länk till Tailwinds webbplats

Tailwind CSS är ett "Utility-first-ramverk. I detta ramverk bygger man sina komponenter själv med olika klasser i HTML. Man har mer kontroll och kan snabbare göra styling utan att man skriver ny CSS. Fördelar med Tailwind är att det är väldigt flexibelt, CSS-filen är välidgt liten, samt kan man göra unik design. Några nackdelar är att det lätt kan bli rörigt med många klasser på de olika HTML-elementen, kräver att man gör "build-steg" och den har ingen färdig design.

Utvärdering av Tailwind

Jag tyckte att Tailwind var rätt enkel att använda, jag installerade den via CLI och föjde instruktionerna på Tailwinds webbplats. Det gjorde installationen smidig och gick bra att förstå. Jag installerade även Tailwind CSS IntelliSense, vilket gjorde det enklare med alla klass-namn. Det gick rätt snabbt att komma igång med att bygga upp en layout och design direkt i HTML-koden. Jag tyckte om de färdiga klasserna som man använder sig av till CSS, så det var smididgt att testa sig fram och få ett resultat direkt.

Det jag tyckte var lite utmanande i början var att få till text-storleken och vissa ställen att få till layouten med utrymme runt om, men efter man jobbat med det en stund så blev det lite tydligare hur man ska tänka. Överlag så tyckte jag att det var ett bra ramverk som man kan få till snygga lösningar.
